## Project Overview
This project is a JavaFX-based Personal Diary Manager that allows users to write, read, search, and manage diary entries through a modern graphical interface.

## Features
- JavaFX GUI with navigation layout
- Rich text diary editor with auto-save
- Entry browser using ListView
- Real-time search functionality
- Light and Dark theme support
- File I/O using Java NIO
- Background tasks for file operations
- User-friendly error handling

## Technologies Used
- Java
- JavaFX
- CSS
- Java NIO (Files API)
- Concurrency (Task, Thread)

## How to Run
1. Open project in VS Code or IntelliJ
2. Ensure JavaFX is properly configured
3. Run `DiaryApp.java`

## Design Choices
- BorderPane for clean layout
- External CSS for easy theming
- Background threads for file I/O to keep UI responsive

## Learning Outcomes
- Advanced JavaFX UI design
- File handling with NIO
- Multithreading in GUI applications
- Exception handling with dialogs
